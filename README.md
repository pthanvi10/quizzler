# 📚 Quizzler - Mini E-Learning App

Quizzler is a simple and interactive **Flutter-based E-Learning & Quiz App** designed to help users learn subjects and test their knowledge through quizzes.

---

## 🚀 Features

* 🧠 Multiple Subjects (Computer Science, Mathematics, History, Chemistry)
* 📖 Study Material for each subject
* ❓ MCQ-based Quiz System
* 📊 Score Tracking using Provider
* 🔄 Restart Quiz functionality
* 🎨 Modern UI with images & colors
* 🖼️ Subject Cards with images

---

## 🛠️ Tech Stack

* **Flutter (Dart)**
* **Material Design 3**
* **Provider (State Management)**

---

## 📂 Project Structure

```
lib/
 └── main.dart   # Complete app (UI + logic)
assets/
 └── images/     # Logo & subject images
```

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone <your-repo-link>
cd quizzler
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the app

```bash
flutter run
```

---

## 📦 Dependencies

Add this in `pubspec.yaml`:

```yaml
dependencies:
  flutter:
    sdk: flutter
  provider: ^6.0.5
  flutter_markdown: ^0.6.18

```

---

## 🧩 How It Works

1. User selects a subject from Home Screen
2. Reads study material
3. Attempts quiz
4. Gets final score
5. Can restart quiz
